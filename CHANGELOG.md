## [5.0.0] - 2024-XX-XX

### Added
- Gestione completa collezione macchine da scrivere e calcolatrici
- Sistema categorie/sottocategorie personalizzabili
- Gestione marchi e modelli gerarchica
- Tracciamento stati funzionali ed estetici
- Gestione accessori con stato disponibilita
- Storico lavori (restauri, riparazioni, manutenzioni)
- Supporto immagini multiple con conversione automatica WebP
- Gestione ubicazioni fisiche
- Dashboard con statistiche collezione
- Import/export JSON completo
- Generazione PDF schede e report
- Interfaccia responsive tema vintage
- Salvataggio automatico localStorage
- Filtri avanzati e ricerca
- Ordinamento personalizzabile

### Technical
- React 18.2 con hooks
- Babel per transpilazione JSX
- jsPDF per generazione documenti
- JSZip per gestione archivi
- CSS custom properties per temi
- Responsive design mobile-first
